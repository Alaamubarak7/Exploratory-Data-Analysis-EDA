Exploratory Data Analysis (EDA)

In this project, I performed Exploratory Data Analysis using SQL on a global layoffs dataset to uncover key patterns and trends in workforce reductions.

Analysis Highlights

• Identified companies with the largest single layoffs and the highest total layoffs.

• Analyzed layoffs by country, location, and industry to see which sectors were most affected.

• Explored layoffs over time (yearly and monthly) to understand trend changes.

• Used CTEs and Window Functions to rank companies with the most layoffs each year.

Tools

SQL | Data Cleaning | EDA | Window Functions | CTEs
